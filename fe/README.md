## 特色
 1. 用户系统

 3. 留言板
 
 
## 安装与运行

拉取仓库

```
git clone https://github.com/Meeken1998/garage-kit-shop shop
cd shop
npm install
```

运行

```
npm run build
```

## 打包
```
npm run build
```
